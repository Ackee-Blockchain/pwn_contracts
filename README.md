# PWN Finance
Smart contracts enabling P2P loans using arbitrary collateral (supporting ERC20, ERC721, ERC1155 standards).

## Developer docs
For in-depth documentation about PWN contracts see [PWN Developer Docs](https://dev-docs.pwn.xyz/).
